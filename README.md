# Trendline Frontend Task

This project is a frontend task built using **Next.js** as part of a technical assessment.

The project includes two main parts:
- Authentication flow (Register, Verify, Login)
- Product Details UI 

---

## Features

### Authentication
- Register page with:
  - Full Name
  - Email
  - Password
  - Phone Number
  - Country Code
- Verify account page  
  (verification code for testing: **123456**)
- Login page
- Token is saved in `localStorage`
- After login, user is redirected to a simple dashboard with welcome message

### UI Pages
- Product Details page
- Rating & Reviews section
- Similar Items section
- Footer
- Fully responsive (Desktop & Mobile)

> Note:  
> UI sections are **static only** as requested.  
> No slider logic, no dynamic color change, just design implementation.

---

## Technologies Used
- Next.js (App Router)
- Tailwind CSS
- Lucide Icons

---

## Getting Started

First, install the dependencies:

npm install

Then, run the development server:

npm run dev

After that, open your browser at:

http://localhost:3000
